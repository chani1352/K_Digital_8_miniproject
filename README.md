## 프로젝트 소개

- 예방이는 신생아부터 유아기, 아동기에 걸친 예방 접종 정보를 확인하고 관리하는 웹 서비스 입니다.
- 시기별, 연령별, 개월수별 필요한 예방 접종 정보를 확인할 수 있습니다.
- 지역을 설정하여 우리 동네에서 백신 접종 가능한 병원 목록을 확인할 수 있습니다.
- 자녀와 자녀의 지난 접종 정보를 기록하며, 앞으로 맞아야할 접종 정보를 스케쥴별로 관리할 수 있습니다.
